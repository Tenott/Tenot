# ALLO MVP (Next.js)

Это MVP сайта объявлений **ALLO — объявления без лишнего**.
Данные (объявления и "авторизация") хранятся в **localStorage** — без сервера.

## Запуск
1) Установи Node.js LTS (18+)
2) В папке проекта:
```bash
npm install
npm run dev
```
3) Открой http://localhost:3000

## Что есть
- Главная, поиск, карточка объявления
- Добавить объявление (с фото-ссылками / загрузка позже)
- "Логин" по телефону (локально)
- Личный кабинет: мои объявления, избранное, настройки
- Простейшая "админка" (очередь/жалобы заглушки)

## Дальше
Легко заменить localStorage на настоящий backend (Nest/Django) и S3-хранилище для фото.
